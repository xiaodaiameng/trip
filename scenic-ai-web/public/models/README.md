# 3D 数字人模型放置说明

本目录用于放置浏览器数字人模型资源。

支持两种方式，按优先级加载：

1. `avatar.glb`
   - 完整数字人模型
   - 放入后，页面优先使用这个完整模型

2. `head.glb`
   - 仅头部模型
   - 当没有 `avatar.glb` 时，会自动替换当前内置数字人的头部

建议：

- 优先使用 `glb` 格式
- 模型朝向尽量为正面朝 +Z 或便于前视展示
- 头部模型中心尽量在脖颈连接处附近，便于替换
- 如果模型包含材质球，系统会尽量保留；若网格命名包含 `hair`、`skin`、`robe`、`sash`、`gold`、`boot`、`gem` 等关键词，会自动参与换肤着色

当前页面路径：

- 模型目录：`/public/models/`
- 运行时访问：
  - `/models/avatar.glb`
  - `/models/head.glb`
